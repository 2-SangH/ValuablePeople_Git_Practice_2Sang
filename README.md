## 목표

----------------------------------------------------------------------------

> ## Contribution 규칙
> 인프런의 강의를 참고하여 다음과 같이 진행
> 1. 프로젝트 fork
> 2. 이슈 등록 - 나는 무엇 무엇을 하겠다. (이를 티켓팅이라고 한다.)
> 3. fork 프로젝트(본인 계정)를 sourcetree로 clone 후 새로운 브랜치를 생성한다.
> 4. PR 요청한다. - PR 요청 시 본인 계정의 fork 프로젝트로 요청한다.
----------------------------------------------------------------------------
